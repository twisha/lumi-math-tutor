What We Built

Problem → Solution

⚠️

The Problem

Young children can't type. Traditional chat tutors are inaccessible to 5–10 year olds who need voice, patience, verified answers — not hallucinated math.

✨

The Solution

Lumi listens via Whisper, thinks with Claude, verifies answers through MCP tools, speaks back via macOS TTS. A real agentic AI loop — end to end.

🎓

Two Grade Levels

K–2: counting, addition & subtraction within 20, with emoji + hop-path visual aids. Grade 3–5: multiplication, division, fractions, multi-step problems.

🛡️

Guardrails

Every input is classified for topic and grade level before responding. Out-of-scope requests get a warm redirect. Answers always tool-verified — never guessed.

AI Engineering Fundamentals

Concepts Applied

✏️

Prompt Engineering

Separate system prompts for K–2 and Grade 3–5 encode persona, teaching strategy, hint escalation, and guardrails entirely in natural language. Zero fine-tuning.

🔧

Agentic Tool Use

Claude autonomously decides when to call calculate(), check_answer(), generate_problem(), check_topic(), or check_grade_level().

🔌

MCP Protocol

Tools run as a local MCP server over stdio. The bridge translates MCP schemas to Anthropic tool format — model-agnostic and fully swappable.

🏷️

LLM Self-Tagging

Lumi appends [COUNT:N] when asking the child to count aloud. The app strips the tag, computes recording duration as N+3 seconds, and adapts — no regex, no rules.

⚡

Prompt Caching

The system prompt is marked ephemeral with Anthropic's cache_control API, reducing latency and token cost on every multi-turn conversation turn.

🎙️

Multimodal I/O

Whisper (transformer STT) with silence-detection early-stop for voice input. macOS say subprocess with Unicode normalization and VAD-aware duration for TTS output.

K–2 Feature Highlight

Context-Aware Visual Aids

🎨

Hop-Path Number Line

Replaces a tiny 0–20 ruler with large 56px circles connected by arrows. ★ START marks the larger operand; yellow circles show each hop; a dashed ? marks where the child must land — answer intentionally hidden.

♿

Color-Blind Accessible

Color is never the only signal. Start circle is larger with a ★ icon and "START" label. Answer slot has a dashed border and "YOU!" label. Any child can follow the path without relying on color.

🧠

Shown Only When Struggling

Visual is hidden on first problem presentation. It appears only when the child asks for help or gives a wrong answer — detected via the child's words and check_answer() tool result.

🔗

Prompt ↔ Visual Alignment

Lumi's spoken hint references the same shapes: "Look at the ★ START circle — hop along each circle and tell me what number you land on at the ? circle." Visual and voice teach the same strategy.
