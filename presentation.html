What We Built

Problem → Solution

⚠️

The Problem

K-2 children can't type. Traditional chat tutors are inaccessible to 5–7 year olds who need voice, encouragement, and verified answers — not hallucinated math.

✨

The Solution

Lumi listens via Whisper, thinks with Claude, verifies answers through MCP tools, and talks back with pyttsx3. A real agentic AI loop — end to end.

🎓

Teaches

Counting, addition, subtraction within 20. Guides with hints, never reveals the answer immediately. Redirects off-topic questions warmly.

🛡️

Guardrails

Classifies every input for topic and grade level before responding. Out-of-scope requests get a warm redirect, never an attempt to answer.

AI Engineering Fundamentals

Concepts Applied

✏️

Prompt Engineering

System prompt encodes Lumi's persona, teaching sequence, hint escalation, and guardrails entirely in natural language. No fine-tuning needed.

🔧

Agentic Tool Use

Claude autonomously decides when to call calculate(), check_answer(), or generate_problem() — never guessing at math.

🔌

MCP Protocol

Tools run as a local MCP server over stdio. The bridge translates MCP schemas to Anthropic format — model-agnostic and swappable.

🛡️

Guardrails

Defense-in-depth: input classification tools and system prompt reinforcement catch off-topic and out-of-grade requests at two layers.

⚡

Prompt Caching

The system prompt is cached with Anthropic's ephemeral cache, reducing latency and cost on every multi-turn conversation turn.

🎙️

Multimodal I/O

Whisper (transformer-based STT) for voice input. pyttsx3 for TTS output. The same transformer architecture powers both the LLM and speech model.
